# Finaltest

Объявляем два массива одинаковой длинны. Потом метод, в котором цикл соразмерный длине массива, внутри цикла проверка условия ( <=3 ), если да, то элемент первого массива заносится в count элемента второго массива. Переменную count используем для последовательной проверки всех элементов. После присвоения увеличивается переменная count на 1 и возвращается к циклу for в котором i увеличивается на 1. И так проверяется до конца.
